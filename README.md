## 🏃 Running Locally

To run this portfolio on your local environment:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vivekfaujdar01/vivek-portfolio.git
   cd vivek-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in the browser:**
   Open [http://localhost:5173](http://localhost:5173) to see your portfolio live.
